Game of Life Parameters and Properties

  • General Parameters
    • Width: width (in number of automata) of the neighbourhood.
    • Height: height (in number of automata) of the neighbourhood.
    • Time scale: controls the iteration computation rate. Expressed as an arbitrary percentage value from 0% (0Hz refresh rate) to 100% (60Hz refresh rate)
  • Game of Life Properties
    • Pattern: selects which Game of Life pattern to execute.
