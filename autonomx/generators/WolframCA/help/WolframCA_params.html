WolframCA Parameters and Properties

  • General Parameters
    • Width: width (in number of automata) of the neighbourhood.
    • Height: height (in number of automata) of the neighbourhood.
    • Speed: controls the rate at which new generations are computed. Expressed as an arbitrary percentage value ranging from 0 (idle) to 100 (60Hz refresh rate).
  • Wolfram Properties
    • Ruleset: selects which Wolfram Code rulset to execute.
    • Random Seed: If selected, the wolfram starts with a random set of alive cells in the first generation. Otherwise, the first generation includes only live cell in the middle.
